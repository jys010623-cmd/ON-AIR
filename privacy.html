개인정보처리방침

시행일: 2026년 6월 18일

ON-AIR(이하 “서비스”)는 개인이 만든 비상업 웹 애플리케이션입니다. 본 방침은 서비스가 이용자의 정보를 어떻게 다루는지 설명합니다.

1. 서비스가 수집·보관하는 정보

서비스는 자체 서버나 데이터베이스를 운영하지 않으며, 이용자의 정보를 수집해 외부로 전송하지 않습니다. 아래 항목은 모두 이용자 브라우저의 로컬 저장소(localStorage)에만 저장되고, 이용자 기기를 떠나지 않습니다.

  • 즐겨찾기, 보관함(저장한 유튜브 링크), 내 플레이리스트, 최근 들은 곡 기록
  • 화면 설정: 기기 스킨, 다크 모드, 볼륨
  • 이용자가 직접 입력한 YouTube Data API 키(선택 사항)

이 데이터는 서비스 운영자를 포함한 누구에게도 전송되지 않으며, 같은 브라우저에서만 유지됩니다.

2. 제3자 서비스

서비스는 다음 외부 서비스를 이용합니다. 해당 서비스에 요청을 보낼 때 IP 주소 등 표준적인 기술 정보가 각 제공자에게 전달될 수 있으며, 이는 각 제공자의 개인정보 정책을 따릅니다.

3. 쿠키

서비스 자체는 추적용 쿠키를 사용하지 않습니다. 다만 유튜브 영상을 재생하면 YouTube가 쿠키를 설정할 수 있으며, 이는 Google의 정책을 따릅니다.

4. 데이터 삭제

이용자는 언제든지 다음 방법으로 저장된 모든 데이터를 삭제할 수 있습니다.

  • 앱 상단의 ‘초기화’ 버튼 사용
  • 브라우저 설정에서 본 사이트의 사이트 데이터/저장소 삭제

5. 아동의 개인정보

서비스는 개인정보를 수집·전송하지 않으므로 특정 연령대를 대상으로 정보를 처리하지 않습니다. 다만 미성년자는 보호자의 지도 아래 이용할 것을 권장합니다.

6. 방침 변경

본 방침은 변경될 수 있으며, 변경 시 본 페이지에 갱신된 내용과 시행일을 게시합니다.
