# 음악 봇 200% 활용 가이드

`index.html`을 제대로 쓰기 위한 두 가지를 정리했어요.
**1번(API 키)** 은 키워드 검색·AI DJ 추천을 켜주고, **2번(로컬 서버)** 은 재생이 끊기지 않게 안정적으로 만들어줘요.
둘 다 무료이고, 합쳐서 10분이면 끝나요.

---

## 1. 유튜브 검색 API 키 발급 (단계별)

키워드로 곡을 검색하거나("뉴진스 검색"), AI DJ가 진짜 추천 셋을 짜려면 이 키가 필요해요.
무료이고 하루 약 100번 검색까지 됩니다(개인용으로 충분).

### 단계
1. **구글 클라우드 콘솔 접속**
   https://console.cloud.google.com 에 구글 계정으로 로그인.

2. **프로젝트 만들기**
   상단 바의 프로젝트 선택 → **새 프로젝트** → 이름(예: `music-bot`) 입력 → 만들기.
   만든 프로젝트가 선택돼 있는지 확인.

3. **YouTube Data API v3 켜기**
   왼쪽 메뉴 **API 및 서비스 → 라이브러리** →
   검색창에 `YouTube Data API v3` 입력 → 클릭 → **사용(Enable)** 버튼.

4. **API 키 만들기**
   왼쪽 **API 및 서비스 → 사용자 인증 정보** →
   상단 **+ 사용자 인증 정보 만들기 → API 키** →
   `AIza...` 로 시작하는 키가 생성됩니다. **복사**하세요.

5. **봇에 넣기**
   `index.html` 을 열고 우측 상단 **검색 설정** →
   복사한 키를 붙여넣고 **저장**. 끝!
   이제 검색창에 "아이유", DJ 칸에 "비 오는 밤 감성" 같은 걸 넣으면 진짜로 검색해 와요.

### (선택) 키 보호하기
공개적으로 쓸 거라면, 사용자 인증 정보에서 만든 키 → **키 제한** →
"애플리케이션 제한"이나 "API 제한(YouTube Data API v3만)"을 걸어두면 안전해요.
혼자 쓸 거면 안 해도 됩니다.

### 참고
- 무료 할당량은 하루 10,000 유닛이고, 검색 1회가 100 유닛이라 **하루 약 100회 검색**까지 가능해요.
- 키는 비밀번호 같은 거예요. 남에게 공유하거나 공개 저장소에 올리지 마세요.

## 2. 안정적인 로컬 서버로 띄우기 (명령어 한 줄)

HTML 파일을 그냥 더블클릭하면 주소가 `file://` 라서, 유튜브 임베드나 검색 호출이
환경에 따라 막힐 수 있어요. 아주 간단한 로컬 서버로 띄우면 주소가 `http://localhost...` 가 되어
모든 기능이 안정적으로 작동해요. **설치할 것 없이 이미 깔린 도구로 한 줄이면 됩니다.**

### 방법 A — 파이썬 (대부분 이미 설치됨, 추천)
이 파일이 있는 폴더에서 터미널(명령 프롬프트)을 열고:

```
python -m http.server 8000
```

> 만약 `python` 이 안 되면 `python3 -m http.server 8000` 으로 시도하세요.

그다음 브라우저 주소창에 입력:

```
http://localhost:8000/index.html
```

### 방법 B — Node.js 가 있다면
```
npx serve
```
실행하면 나오는 주소(예: `http://localhost:3000`) 뒤에 파일 이름을 붙여 접속하면 돼요.

### 정리
- 서버를 켠 터미널 창은 사용하는 동안 **켜둔 채로** 두세요.
- 끄려면 터미널에서 `Ctrl + C`.
- 매번 켜기 번거로우면, 위 명령을 적은 `.bat`(윈도우) / `.command`(맥) 파일을 만들어 더블클릭으로 실행해도 돼요.

## 둘을 합치면
1번으로 키를 넣어 검색·AI DJ를 켜고 → 2번으로 `localhost` 에서 열면,
기기 스킨(LP·CD·벽걸이 CD·카세트·스마트 스피커·턴테이블 세트·아이팟·MP3·PMP)을 바꿔가며
끊김 없이 검색·재생·가사·즐겨찾기까지 전부 매끄럽게 쓸 수 있어요.

막히는 단계가 있으면 그 화면을 캡처해서 보여주세요. 바로 같이 풀어드릴게요!
